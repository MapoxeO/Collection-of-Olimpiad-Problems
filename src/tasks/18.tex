\task {
    \newtaskref{complex}
} {
    Докажите, что при некотором \(\lambda \in \mathbb{R}\) три попарно различных комплексных числа $a,b,c\in\mathbb{C}$ такие, что $\abs{a}=\abs{b}=\abs{c}=1$, удовлетворяющие уравнению
    $a^3+b^3+c^3=\lambda \cdot abc$, образуют равнобедренный треугольник в комплексной плоскости.
    Найдите все значения параметра \(\lambda\), при котором верна гипотеза выше, а также найдите угол \(\theta\) при вершине равнобедренного треугольника.
} {
    РЕШЕНИЕ
} {}
