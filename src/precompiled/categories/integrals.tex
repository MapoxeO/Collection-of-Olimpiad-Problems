\section{Интегралы}
\hyperlink{integrals:1} {\large{\textbf{ Задача 1(7)}}}
Решить дифференциальное уравнение
    $$\int \limits_{0}^{\frac{dy}{dx}} { \frac{\cos t\  dt}{16 + 9 \sin^2 t} } = \frac1{12}\arctg{x}$$\\[4mm]
\hyperlink{integrals:2} {\large{\textbf{ Задача 2(8)}}}
Вычислить $$ \iint \limits_D \frac{dxdy}{x^3+y^3},\quad \text{где }D \colon\
    \begin{cases}
        x + y \geqslant 1 \\
        x \geqslant 0 \\
        y \geqslant 0
    \end{cases}$$\\[4mm]
\hyperlink{integrals:3} {\large{\textbf{ Задача 3(9)}}}
Вычислить определенный интеграл: $\hspace{4mm} \displaystyle \int \limits_0^1 \arcsin{x} \arccos{x} \ dx$\\[4mm]
\hyperlink{integrals:4} {\large{\textbf{ Задача 4(15)}}}
Вычислить двойной интеграл $\displaystyle \iint\limits_{D}\abs{\cos{(x+y)}}dS$, где $D\colon\;  0 \leqslant x+y\leqslant \pi \; \land\; xy \geqslant 0$\\[4mm]
\hyperlink{integrals:5} {\large{\textbf{ Задача 5(20)}}}
Найти наибольшую площадь круга, вписанного в ограниченную область $D$, такую что 
    $\forall (x, y) \in D \Rightarrow \\ x^3~+~y^3~\leqslant~x^2y^2~+~xy$.\\[4mm]
\hyperlink{integrals:6} {\large{\textbf{ Задача 6(21)}}}
Доказать сходимость интеграла, и если он сходится, найти его значение
    \[
        \int_0^{1} \lr{\frac{1}{x}-\floor{\frac{1}{x}}}dx
    \]\\[4mm]
